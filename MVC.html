What is MVC

Model–view–controller (MVC) is a software architectural pattern for implementing user interfaces.

Advantages of MVC

- ASP.NET MVC framework doesn’t use View State and thus reduces the bandwidth of the requests to an extent.
- The MVC framework provides a clean separation of the UI, Business Logic, Model or Data.

MVC application life cycle

- Routing

- Controller
- Action Execution
- View Result
- View Engine
- View

Routing/Route

A route is a URL pattern.
a) Convention-based routing - to define this type of routing, we call MapRoute method and set its unique name, URL pattern and specify some default values.
b) Attribute-based routing - to define this type of routing, we specify the Route attribute in the action method of the controller

Temp data, View, and View Bag

ViewData:
- ViewData is used to pass data from controller to view.
- Type is object
- Requires typecasting.
- If redirection occurs, then its value becomes null.

ViewBag:
- ViewBag is also used to pass data from the controller to the respective view.
- Type is a dynamic
- Doesn’t require typecasting
- If redirection occurs, then its value becomes null.

TempData
- TempData is used to pass data from the current request to the next request
- It keeps the information for the time of an HTTP Request. This means only from one page to another. It helps to maintain the data when we move from one controller to another controller or from one action to another action

Partial View in MVC

Partial view is a reusable view (like a user control) which can be embedded inside another view.

Areas in MVC

Areas are just a way to divide or “isolate” the modules of large applications in multiple or separated MVC

Data Annotation in MVC

DataAnnotation used for validation to properties while designing the model itself. This validation can be added for both the client side and the server side.
Ex: Required,Range,StringLength

Bundling

Combine or bundle multiple files into a single file.
Combine multiple JavaScript (.js) files or multiple cascading style sheet (.css) files so that they can be downloaded as a unit, rather than making individual HTTP requests.

Minification

Removing unnecessary white space and comments to reducing the size to improve request load time.
