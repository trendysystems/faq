SQL

SQL stands for Structured Query Language , and it is used to communicate with the Database. This is a standard language used to perform tasks such as retrieval, updation, insertion and deletion of data from a database.

Primary Key

(a) A Primary key is a column (or collection of columns) or a set of columns that uniquely identifies each row in the table.
(b) Null values not allowed
(c) Only one Primary key constraint defined per table.

Unique key

(a) A Unique key constraint uniquely identified each record in the database. This provides uniqueness for the column or set of columns.
(b) Null values allowed but only one
(c) May Unique key constraint can be defined per table.

Foreign key

A foreign key is one table which can be related to the primary key of another table. Relationship needs to be created between two tables by referencing foreign key with the primary key of another table.

What are Constraints

Constraints are used to specify the limit on the data type of the table. It can be specified while creating or altering the table statement. The sample of constraints are:

NOT NULL
CHECK
DEFAULT
UNIQUE
PRIMARY KEY
FOREIGN KEY

What is a join

This is a keyword used to query data from more tables based on the relationship between the fields of the tables.

Type of Joins?

a) JOIN OR INNER JOIN: Returns records that have matching values in both tables
b) LEFT JOIN OR LEFT OUTER JOIN: Returns all records from the left table, and the matched records from the right table
c) RIGHT JOIN OR RIGHT OUTER JOIN: Returns all records from the right table, and the matched records from the left table
d) FULL JOIN OR FULL OUTER JOIN: Returns all records when there is a match in either left or right table
e) CROSS JOIN: produces a result set which is the number of rows in the first table multiplied by the number of rows in the second table
f) SELF JOIN: A self JOIN is a regular join, but the table is joined with itself.

What is the difference between DELETE and TRUNCATE and DROP statements

Delete
- Delete command is used to delete a row in a table as per the condition.
- You can rollback data after using delete statement.
- It is a DML command.
- It is slower than truncate statement.

Truncate
- Truncate is used to delete all the rows from a table.
- You cannot rollback data.
- It is a DDL command.
- It is faster.

DROP
- It is a DDL command.
- This command will drop the table data and structure.

DDL, DQL, DML, DCL and TCL

DDL:
- DDL is Data Definition Language and is used to define the structures like schema, database, tables, constraints etc.
- Examples of DDL are CREATE, DROP, RENAME and ALTER.
DML:
- DML is Data Manipulation Language and is used to manipulate data.
- Examples of DML are INSERT, UPDATE and DELETE.
DQL:
- Data Query Language
- DML statements are used for performing queries on the data within schema objects.
- Examples: SELECT
DCL:
- Data Control Language
- DCL includes commands such as GRANT and REVOKE which mainly deals with the rights, permissions and other controls of the database system.
- Examples
GRANT - gives user’s access privileges to database.
REVOKE - withdraw user’s access privileges given by using the GRANT command.
TCL:
- Transaction Control Language
- TCL commands deals with the transaction within the database.
Examples
COMMIT – commits a Transaction.
ROLLBACK – rollbacks a transaction in case of any error occurs.
SAVEPOINT – sets a savepoint within a transaction.
SET TRANSACTION – specify characteristics for the transaction.

Normalization vs Denormalization

- Normalization is used to remove redundant data from the database by using forign key relationship.
- Denormalization is used to combine multiple table data into one so that it can be queried quickly.

Stored Procedures

- The group of SQL statements which are or have been stored in the server database is known as a stored procedure.
- Procedures are used to process the huge amount of data for complex procedures and functionalities.
- it is faster because it is a pre-compiled block of code.

UNION VS UNION ALL

Both union and union all used to combine two sets of results into one result.
But UNION removes duplicate values

What is an aggregate function

An aggregate function is an SQL function that reads data from multiple rows and displays a single value. Some examples of aggregate functions are COUNT, SUM, MIN, MAX, and AVG. They are often used with a GROUP BY clause but can be used by themselves.

What is Cursor

A Cursor is a database object that represents a result set and is used to manipulate data row by row.

What is View

- A view is a virtual table which consists of a subset of data contained in a table.
- View can have data of one or more tables combined, and it is depending on the relationship.
